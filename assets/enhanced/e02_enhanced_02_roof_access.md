# 1층 지붕 접근 가능성 검토

## 시뮬레이션

### 시나리오 A: 사다리 사용
- 필요 높이: 2.8m
- 일반 가정용 사다리로 충분
- **문제: 사다리 발견 안 됨**

### 시나리오 B: 의자 사용
- 정원 테이블 의자: 높이 45cm
- 의자 + 점프: 약 3.0m 도달 가능
- **가능성: 높음**
- 정원에 눌린 자국과 일치!

### 시나리오 C: 담장 이용
- 담장 높이 2.5m
- 담장 → 1층 지붕 점프: 가능
- **문제: 담장에 오른 흔적 없음**

## 1층 지붕 → 2층 창문

### 높이 차이: 2.4m
- 맨손으로: 불가능
- 로프 사용: 가능
- 창틀에 고정 → 올라가기

### 문제점
**창문이 내부 잠김 상태!**
- 외부에서 어떻게 잠갔는가?
- 내부에 공범?
- 아니면 트릭?
